BAT2EXE
A file to convert (.bat) file into EXE
Support drag and drop ex. Drag ABC.bat file and drop over BAT2EXE.bat and  ABC.exe will be created in the same directory of where the BAT2EXE.bat is

VB2EXE
To convert ABC.vb to ABC.exe
This script has to be modified to the exact location and name of the desired file and the desired exe file will be created in the same directory as the VB2EXE.bat is
